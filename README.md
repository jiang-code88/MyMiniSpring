# MyMiniSpring
A project of mini spring



## Commit message standard 

```
<date>-<type>: <body>
```

for example

```
# 文档修改
23/3/25-doc: update Readme document
# 代码修改
23/3/25-code: fix bug of spring.java
```

## Directory tree 

```
notes # 目录存放笔记
README.md # 文件 readme 信息
.gitignore # 文件 指定 git 提交忽略的文件
```

> 如果进行了项目目录变更，请记得在 README 文件中更新目录树信息。
>
